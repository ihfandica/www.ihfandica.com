Sistem Pengocok Arisan Digital

Unggah file arisan Anda, kelola riwayat secara otomatis, dan unduh data pencadangan dalam satu file tunggal. Jika tidak punya data (.txt) silakan ketik manual di kotak yang tersedia. Semua proses dilakukan di lokal, tidak ada data yang di unggah. :-)

Sebelum keluar, unduh data terlebih dahulu agar dapat digunakan kembali di masa mendatang tanpa memasukan data ulang.
Lihat langkahnya
  • Pilih file arisan .txt jika sudah punya. Apabila belum, masukkan manual dalam kotak "Pratinjau File / Edit Manual:".
  • Setelah data di pilih, maka kotak akan terisi otomatis. Setelah kotak terisi, pastikan kembali datanya untuk yang belum di kocok atau sudah di kocok. Lihat kotak!
    "== SUDAH KELUAR =="
    adalah batas untuk bagian atas yang belum di kocok dan bagian bawah yang sudah di kocok.
  • Tekan tombol "Mulai Kocok Arisannya" dan tunggu.
  • Jika sudah di lakukan, pastikan unduh datanya agar di masa depan tidak perlu mengisi ulang manual.
  • Isi kotak pengecualian pemenang dengan angka jika ingin nama-nama yang telah keluar tidak keluar kembali. Aturannya adalah nama yang di kocok identik. Contoh data dalam kocokan:
    Nama Peserta
    ABC
    ABC 1
    ABC 2
    DEF
    DEF 1
    DEF 2
    GHI
    GHI 1
    GHI 2
    JKL
    JKL 1
    JKL 2
    Anggaplah ABC dan DEF sudah masuk daftar
    "== SUDAH KELUAR =="
    (terdapat 2 nama):
    Nama Peserta
    ...
    ...
    == SUDAH KELUAR ==
    ABC
    DEF
    Maka jika kotak di isi angka 2, hasilnya nama ABC dan DEF tidak akan di kocok kembali. Apabila dalam satu periode terdapat dua kali kocokan, setelah kocokan pertama angkanya dapat di naikkan jadi 3 (2 untuk mengecualikan ABC dan DEF, 1 lagi untuk mengecualikan hasil kocokan terbaru).
Reset: Mengembalikan data yang telah terkocok menjadi tidak terkocok semua.
Clear: Menghapus semua data, termasuk yang telah di pilih melalui menu Pilih File Arisan .txt.
